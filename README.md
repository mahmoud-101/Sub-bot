# Video → AI Skill

أداة بتحول فيديو أو محاضرة إلى Skill معرفية تقدر تكلمها بعدين.

## الوظائف
- رابط فيديو → Transcript مباشر لو متاح.
- لو مفيش Transcript: تنزيل الصوت بـ yt-dlp ثم تفريغه بـ Whisper.
- تقسيم المحتوى مع timestamps.
- Embeddings للبحث الدلالي.
- إنشاء Skill محفوظة محلياً.
- سؤال وجواب من محتوى الفيديو مع إظهار المقاطع والتوقيتات الداعمة.
- لو السؤال مش متغطي، الـSkill مطالبة تقول كده بدل اختراع إجابة.

## التشغيل
محتاج Python 3.10+ وFFmpeg.

```bash
pip install -r requirements.txt
streamlit run app.py
```

بعد فتح التطبيق، حط OpenAI API Key في الشريط الجانبي، الصق رابط الفيديو، واضغط إنشاء Skill.

## ملاحظات
استخدم فقط محتوى عندك حق الوصول ليه. بعض المواقع المحمية بتسجيل دخول أو DRM قد لا تعمل مباشرة. الـSkills بتتحفظ حالياً في مجلد `skills/` كملفات JSON. النسخة الحالية MVP، والخطوة التالية دعم كورس كامل من عدة فيديوهات وقاعدة بيانات وواجهة إدارة أفضل.
